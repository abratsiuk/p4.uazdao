## ðŸŽ¯ Description

* **Uazdao** is a practical philosophy focused on how to work with complex systems in real life.
* The author, **Pavel Ievlev**, is a science fiction writer and editor of an automotive website.
* In the books, the **UAZ-469** becomes a symbol of meditation and system thinking:

  * It always moves
  * It always breaks
  * It can always be repaired
  * But it can *never* be fully fixed
* The project visualizes and organizes this philosophy in a friendly, structured interface.
* The book belongs to a broader sci-fi universe spanning multiple worlds and timelines.
